Question 3

Refer to the exhibit. When PC1 sends a packet to PC2, the packet has which source and destination IP address when it arrives at interface Gi0/0 on router R2?

source 10.10.1.1 and destination 10.10.2.2
source 192.168.10.10 and destination 192.168.20.10

The correct answer is source 192.168.10.10 and destination 192.168.20.10.
